# 2026-04-19 PC中控 UI 设计

> 状态: 定稿  最后更新: 2026-04-19  对话: [链接]

## 对话目标
PC中控主平台 UI 完整原型设计，含登录窗口 + 四页主平台

## 关键结论

1. **窗口规格确认**
   - 登录窗口：360×300 逻辑像素（PySide 6 固定尺寸）
   - 主平台：1800×1200 逻辑像素，最小可用尺寸 1280×800
   - 设置弹窗：1200×900

2. **DPI 缩放方案（已确认可行）**
   - `os.environ["QT_AUTO_SCREEN_SCALE_FACTOR"] = "1"` 写在 QApplication 创建之前
   - `HighDpiScaleFactorRoundingPolicy.PassThrough` 解决非整数倍模糊
   - 代码全程只用逻辑像素，Qt 自动处理物理像素

3. **Logo 规格**
   - SVG 优先；备用 @1x 300×80 / @2x 600×160 / @3x 900×240 PNG
   - `devicePixelRatio` 运行时判断加载哪个 fallback

4. **账号信息显示（顶栏）**
   - 可激活数 / 已激活数 / 未激活数 / 到期时间
   - 到期时间 < 30天时高亮橙色

5. **设备超级列表设计**
   - 左侧设备面板与右侧表格合并为单一大列表
   - 双击行 → 设备编辑弹窗
   - 右键菜单 → 启动/停止/重启脚本、激活设备、编辑、查看日志、解绑
   - 激活设备通过 ADB Shell 命令执行（见激活方案文档）
   - 列：编号、序列号、角色、状态、激活状态、当前任务、等级、战力、区服、心跳、备注

6. **勾选批量操作**
   - 勾选后可批量：角色、区服、脚本操作、激活模式
   - `QTableWidget.setSelectionMode(ExtendedSelection)` 支持 Ctrl/Shift 多选

7. **全局设置弹窗（8个标签页）**
   - 账号设置、仓库设置、物品设置、制造设置、活动设置、铸币设置、任务设置、其他设置

8. **四大功能页面确认**
   - 设备管理（含超级列表）
   - 组队管理（队伍卡片 + 进度条）
   - 同服订单（交易卡片）
   - 跨服价格聚合（多区服比价表格，最低价自动标绿）

## 代码产出
- `main_platform_v1.html` — 初版原型（登录+主平台四页）
- `main_platform_v2.html` — 完整版原型（账号信息+合并列表+批量+激活+设置弹窗）
- 两个文件暂存本地，待拆分为 PySide 6 正式代码后提交 GitHub
