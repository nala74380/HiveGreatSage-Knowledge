## 当前已确认

- AndroidScript 已接入 Verify 登录、心跳、参数同步、热更新检查和热更新下载相关模块。
- `脚本/framework/updater.lua` 已同步 Verify 当前 `current_version` / `release_notes` 字段契约。
- `脚本/framework/updater.lua` 已取消“安装前写入 hive_script_version”的旧风险口径。
- Verify 源码层已将热更新客户端 `check` / `download` 读取统一到主库 `VersionRecord`。

## 当前待验证

- 懒人精灵环境中的文件级 SHA-256 校验能力。
- `installLrPkg` 在目标设备、目标打包方式下的实际返回值语义。
- AndroidScript 实机下载、安装、重启、新版本识别闭环。
- Updater 遇到 401 后是否需要统一走 `Verify.ensure_token()` 后重试。
- PCControl 是否能在热更新后继续正确识别安卓端上报设备。

参见 [[待决策清单]] T012–T017 以及热更新新增待决策项。
