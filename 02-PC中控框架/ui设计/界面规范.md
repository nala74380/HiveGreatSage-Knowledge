# PC中控 UI 规范

> **当前定位：** 本文是 PC 中控 UI 规范草稿入口。
> **历史说明：** 本文保留早期 UI 设计结论，部分内容来自历史 AI 对话提炼。
> **证据边界：** 本文不代表当前 PCControl UI 源码、样式文件或打包结果已完成复核。

## 当前证据边界

- 本文当前作为 D1 级 UI 规范文档维护。
- 本轮未复核 PCControl 当前 main 的 PySide6 UI 源码。
- 本轮未复核 QSS 样式文件。
- 本轮未完成真实分辨率、DPI、窗口缩放或多屏测试。
- 不得把本文中的尺寸、色彩、页面结构写成"已实现 / 已验证 / 生产可用"。
