## 一、总体结论

已确认：前端已从早期页面进入多角色后台阶段，采用 Vue 3 + Vite + Element Plus + Pinia + Axios。主路由已经直接写在 `frontend/src/router/index.js`，`router/routes/admin.js/common.js/shared.js` 仍存在，但当前主入口并未按旧拆分方式组合这些路由文件。

待确认：需要运行 `npm install` 与 `npm run build` 验证当前前端是否可构建。

- 已确认：存在旧拆分路由文件与当前 `router/index.js` 内联路由并存现象。建议清理为一种路由组织方式，避免后续开发者误改未生效文件。
- 待确认：页面是否全部接上当前后端接口，需要逐页打开验证。
- 建议方案：新增 `前端页面-后端接口对照表.md`，每个页面列出读取接口、写入接口、权限角色、空态和错误态。
