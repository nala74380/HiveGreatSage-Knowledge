## 一、结论

✅ 旧点数系统已全量删除。新账务中心为唯一真相源。

旧系统仅保留本文档作为历史迁移说明。

## 二、旧系统对象

| 旧对象 | 当前状态 |
|---|---|
| `AgentBalance` | 已退出 ORM 主线 |
| `BalanceTransaction` | 已退出 ORM 主线 |
| `AuthorizationCharge` | 已退出 ORM 主线 |
| `balance_admin.py` | 当前 main.py 未注册旧 balance_admin 路由；旧文件物理存在状态本轮未确认 |
| `balance_agent.py` | 当前 main.py 未注册旧 balance_agent 路由；旧文件物理存在状态本轮未确认 |

| 当前对象 | 用途 |
|---|---|
| `AccountingWallet` | 代理钱包 |

| `AccountingReconciliationRun` | 对账批次 |
| `AccountingAdjustmentRequest` | 调账申请 |
| `AccountingRiskEvent` | 风控事件 |

```text

/admin/api/prices/*
/api/agents/my/balance

```

```text

禁止在 Knowledge 中继续写“旧点数兼容层已保留”。
```
