# 旧字段旧接口清理清单


## 旧字段

| 对象 | 旧字段 | 当前风险 | 清理策略 | 状态 |
|---|---|---|---|---|
| User | user_level | 与 Authorization.user_level 冲突 | 停止业务读取 | ✓ ORM 模型已无此字段 |
| User | max_devices | 与 Authorization.authorized_devices 冲突 | 前端和文档先改 | ✓ ORM 模型已无此字段 |
| User | expired_at | 与 Authorization.valid_until 冲突 | 登录链路不应全局拦截 | ✓ ORM 模型已无此字段 |
| Agent | level | 组织层级易与业务等级混淆 | 文档统一写 org_level / tree_level 语义 | 待重命名 |
| AgentBalance | 整表 | 与 AccountingWallet 并存 | 迁移后删除 | ✓ 已删除 |
| BalanceTransaction | 整表 | 与 AccountingLedgerEntry 并存 | 迁移后删除 | ✓ 已删除 |
| AuthorizationCharge | 整表 | 与 AuthorizationChargeSnapshot 并存 | 迁移后删除 | ✓ 已删除 |

| 接口或文件 | 状态 | 建议 | 执行结果 |
|---|---|---|---|
| balance_admin.py | 旧管理端点 | 拆分到 pricing.py + accounting.py | ✓ 已完成 |
| balance_agent.py | 旧代理端点 | 合并到 agents.py | ✓ 已完成 |
| `/admin/api/prices/*` | 项目定价 | 独立为 pricing.py 路由 | ✓ 已完成 |
| `/admin/api/agents/{agent_id}/balance` | 旧余额 | 删除，用 accounting.py /wallets/{agent_id} | ✓ 已删除 |
