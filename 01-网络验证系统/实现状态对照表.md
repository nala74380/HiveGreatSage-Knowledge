# 实现状态对照表


## 状态分级

| 状态 | 含义 |
|---|---|
| 源码已存在 | 静态源码中已看到模型、路由、服务或页面实现 |
| 待运行验证 | 需要实际启动、迁移、测试或构建确认 |
| 待三端联调 | 需要 Verify + PCControl + AndroidScript 联调 |
| 待生产化 | 需要部署、安全、压测、监控、回滚闭环 |
| 待修订 | 文档或代码口径已发现漂移 |
