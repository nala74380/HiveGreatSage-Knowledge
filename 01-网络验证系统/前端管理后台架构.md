# 前端管理后台架构


## 当前结论

已确认：Vue 管理后台已经覆盖管理员、代理、项目、设备、账务、系统设置、热更新等主要页面。

## API 分层

- `src/api/admin/*`：管理员后台接口。
- `src/api/agent/*`：代理端接口。
- `src/api/client/*`：终端 / 设备接口。
- `src/api/shared/*`：共享接口。
- `src/api/accounting.js`：新账务中心主线。



## 设备轮询 ()

10 秒间隔轮询设备列表，支持三种身份自动降级：
1. Admin/Agent →  (page_size=200)
2. 401 或无权限 → 回落  (User Token)
3. 指定 userId →  (单用户绑定)

错误不丢旧数据（保留上一次成功加载的列表），组件卸载时自动清除定时器。

## 认证安全

- Token 仅存 sessionStorage（标签页级），关闭标签页清空
- 跨标签页登出同步： 事件广播 +  监听
- 401 自动跳 ，403 跳 
- 管理员/代理分别鉴权路由守卫
